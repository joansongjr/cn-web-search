# CN Web Search - 中文网页搜索

多引擎聚合搜索，**内置防封策略 + 备用源**，中文搜索更稳定。零配置、无需 API Key。

## 核心优势

- 🛡️ **防封策略** — 请求间隔 + 自动切换备用源
- 🇨🇳 **中文优化** — 360 搜索 + 搜狗 + 必应三重保障
- 🌍 **英文支持** — DDG + HN + Reddit + ArXiv
- 📦 **零依赖** — 纯 SKILL.md
- 🔒 **安全** — 无可执行代码，不会被 VirusTotal 标记

## 搜索引擎

| 引擎 | 状态 | 说明 |
|------|------|------|
| 360 搜索 | 主引擎 | 中文首选，可能被限 |
| 搜狗网页 | 备用1 | 360 被拦截时自动切换 |
| 必应中文 | 备用2 | 最稳定，最后防线 |
| DuckDuckGo | 主引擎 | 英文通用搜索 |
| Hacker News | 专用 | 科技/编程趋势 |
| Reddit | 专用 | 社区讨论 |
| ArXiv | 专用 | 学术论文 |

## 防封策略

### 请求间隔
- 每次搜索后 **等待 3-5 秒**
- 连续请求不超过 3 次
- 超过后等待 30 秒

### 备用源切换
```
360 被拦截 → 搜狗 → 必应
```
