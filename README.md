# CN Web Search - 中文网页搜索

**12+ 搜索引擎聚合**，最全免费搜索 skill。零配置、无需 API Key。

## 核心优势

- 🛡️ **防封策略** — 请求间隔 + 多备用源
- 🔄 **12+ 引擎** — 中文/英文/技术/投资/学术全覆盖
- 🇨🇳 **中文优化** — 360/搜狗/必应/东方财富/财新
- 🌍 **英文支持** — Qwant/Startpage/DDG/必应
- 💻 **技术专用** — Stack Overflow + GitHub Trending
- 📚 **学术** — ArXiv
- 🧠 **知识** — Wolfram Alpha ⭐
