# CN Web Search - 中文网页搜索

**13+ 搜索引擎聚合**，最全免费搜索 skill。包含公众号文章搜索！

## 核心优势

- 🛡️ **防封策略** — 请求间隔 + 多备用源
- 🔄 **13+ 引擎** — 中文/英文/技术/投资/学术/公众号全覆盖
- 🇨🇳 **公众号搜索** — 搜狗微信 + 必应索引 ⭐
- 🇨🇳 **中文优化** — 360/搜狗/必应/东方财富/财新
- 🌍 **英文支持** — Qwant/Startpage/DDG/必应
- 💻 **技术专用** — Stack Overflow + GitHub Trending
- 🧠 **知识** — Wolfram Alpha

| 类别 | 引擎 | 说明 |
|------|------|------|

| 中文 | 360/搜狗/必应 | 主+备用 |

| 深度 | 财新 | 财经报道 |
